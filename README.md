## :gear: Technologies

This project was developed made by me to my class of initial concepts in React, with the following technologies:

  - [ReactJS](https://reactjs.org/)
  - Vite
  - CSS
  - Styled Components
  - TypeScript
  - Axios
  - React Router Dom
  - Zod
  - React Hook Form
  
  And this concepts:
  
   - Props
   - Components
   - Modules
   - Context
   - Hooks
   - State

## :computer: How to run

  ```bash
    # clone repository
    git clone

    # enter web folder
    cd project-name

    # download dependencies
    yarn or npm install

    # running web project
    yarn or npm run dev
  ```
